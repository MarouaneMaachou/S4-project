# Usage

1 - Run the program :

```python3 main.py```

2 - Select location of the face-database directory (make sure to select the directory by double-clicking it)

3 - Annotate photos via click on the buttons or using the left and right mouse button

4 - Quit and save results using the button

5 - Export the results that are in 'results.pkl', in the same directory as main.py
